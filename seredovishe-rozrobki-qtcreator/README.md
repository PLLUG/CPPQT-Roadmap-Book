# Середовище розробки QtCreator

Для розробки програм з використанням бібліотеки Qt було створене інтегроване середовище розробки Qt Creator. Його перша версія була презентована одночасно з офіційним випуском Qt 4.5.0. Це повноцінне кросплатформне середовище для створення нових проектів та роботи над ними. Наразі ми будемо використовувати середовище QtCreator для роботи зі звичайними C++ проектами. У наступних розділах ми також будемо використовувати це середовище для розробки Qt проектів з графічним інтерфейсом.

Ми розглянемо роботу з середовищем Qt Creator версії 4.9.0, яке дозволяє керувати цілим рядом етапів розробки програми, такими як: керування сесіями та проектами, редагування та створення програмного коду, конструювання користувацького інтерфейсу програми, аналіз швидкодії, аналіз використання ресурсів, відлагодження, побудова проекту, запуск програми.
